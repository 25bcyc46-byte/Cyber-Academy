[ ACTIVITY 03 ‚Äî CASE STUDIES ]

Real-World Breach Analysis

2017

WannaCry Ransomware

Affected 200,000+ computers across 150 countries. Learn how a single unpatched vulnerability led to billions in damage.

RansomwarePatch ManagementNHS Impact

Attack Vector: EternalBlue SMB exploit (leaked NSA tool) targeting unpatched Windows XP systems.

Lesson: Always apply security patches. Legacy system management is a critical security concern.

Mitigation: Segment networks, maintain offline backups, enforce patch policies.

2020

SolarWinds Supply Chain

Nation-state attack compromised software build pipeline, affecting 18,000 organizations including US government agencies.

Supply ChainNation-StateAPT

Attack Vector: Malicious code injected into SolarWinds Orion software updates, distributed to customers.

Lesson: Third-party software risk is real. Zero-trust architecture and software bill of materials (SBOM) are critical.

Mitigation: Vendor security assessments, code signing, behavioral monitoring.

2021

Colonial Pipeline Attack

Ransomware on OT systems caused fuel shortages across US East Coast. A single compromised VPN credential caused national impact.

Critical InfrastructureOT SecurityCredential Theft

Attack Vector: Compromised VPN credentials (no MFA). DarkSide ransomware deployed on corporate IT network.

Lesson: MFA is non-negotiable. IT/OT network segmentation is critical for critical infrastructure.

Mitigation: Enforce MFA on all remote access, segment operational technology networks.
